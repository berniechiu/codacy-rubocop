
Checks for optional arguments to methods
that do not come at the end of the argument list.

# Examples

```ruby
# bad
def foo(a = 1, b, c)
end

# good
def baz(a, b, c = 1)
end

def foobar(a = 1, b = 2, c = 3)
end
```

[Source](http://www.rubydoc.info/gems/rubocop/RuboCop/Cop/Style/OptionalArguments)
